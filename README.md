## 🌟 核心理念 (Core Philosophy)

**PureScholar** 剔除了现代网页中冗余的交互和沉重的视觉装饰，回归学术本质。它旨在让你的论文、研究项目和学术动态成为网页的绝对主角。

* **纯粹 (Pure)**：干净的排版，极高的信噪比。
* **专业 (Professional)**：预设学术模块，符合主流学术审美。
* **持久 (Persistent)**：基于静态网页技术，链接永久有效，易于被 Google Scholar 索引。

---

## ✨ 功能特性 (Features)

* **⚡️ 快速响应**: 极致的代码优化，极速加载，全设备自适应布局。
* **📄 论文管理**: 支持 BibTeX 引用，一键展示 PDF、代码仓库、项目主页链接。
* **🕒 动态更新**: 优雅的 `News` 列表，实时分享你的学术进展。
* **🛠️ 模块化设计**: 
    * **Biography**: 个人简介与研究兴趣。
    * **Publications**: 结构化的研究成果展示。
    * **Projects**: 展示 LogicVerse 等深度开发项目。
    * **Services**: 审稿、讲座及学术志愿者经历。
* **🎨 定制化**: 支持轻量级的色彩主题自定义（只需修改一个变量）。

## 🚀 3 分钟快速开始 (Quick Start)

### 1. 部署到 GitHub Pages
1.  **Fork** 本仓库。
2.  进入仓库设置 (`Settings > Pages`)，确保 `Branch` 设置为 `main`。
3.  重命名仓库为 `你的用户名.github.io`。

### 2. 本地预览
确保你已安装 `Ruby` 和 `Jekyll`:

```bash
# 安装依赖
bundle install

# 启动本地服务器
bundle exec jekyll serve
````

访问 `http://localhost:4000` 即可查看。
